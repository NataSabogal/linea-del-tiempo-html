1969

El 29 de octubre de 1969, el primer mensaje se envió a través de ARPANET, la red precursora de Internet. El mensaje fue "LO", que se suponía que era "LOGIN", pero el sistema se cayó después de las primeras dos letras.

1983

El 1 de enero de 1983, ARPANET adoptó el protocolo TCP/IP, lo que marcó el nacimiento oficial de Internet. Este cambio permitió la interconexión de diferentes redes y sentó las bases para la expansión global de Internet.
