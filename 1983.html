Protocolo TCP/IP

1983

En 1983 se adoptó oficialmente el protocolo TCP/IP como estándar para la comunicación entre redes. Este conjunto de protocolos permitió que distintas redes pudieran interconectarse y comunicarse entre sí

Este cambio marcó el nacimiento formal del Internet moderno, ya que estableció las reglas fundamentales para la transmisión de datos que aún se utilizan en la actualidad
